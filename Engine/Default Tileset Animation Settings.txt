---- DEFAULT TILESET ANIMATION SETTINGS ----
Here are all the settings for objTileAnimation for all the default tilesets. Feel free to use these in your level.
Each of these have to be on seperate layers. Change animationLayer to whatever layer you're putting the tile on.
Use Ctrl+F to find the tileset you're looking for quickly.

This is incomplete, but it's...something.
